This mod provides a fix to loss of mutagens upon starting NG+ and also ensures mutagens stack in NG+.
To preserve mutagens going into NG+, please ensure that this mod is installed before starting NG+.
All other behavior is completely unchanged (tested on 1.31).

Installation:
Copy the "modNgPlusMutagenStackingFix" folder to your "The Witcher 3 Wild Hunt\mods" directory.

Uninstallation:
Remove "modNgPlusMutagenStackingFix" folder from your "The Witcher 3 Wild Hunt\mods" directory.
